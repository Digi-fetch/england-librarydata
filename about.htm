about this

Public library data is rich, varied, interesting, and there is a great demand for it. But with more public open data available than ever before, used for creative and analytical purposes, library data is also very rare.

Other than citizen personal information, there is no good reason to not make all data related to public libraries openly licensed, freely available for anyone to use.

The Libraries Taskforce for England have released their core libraries dataset, compiled and then validated by the 151 library authorities in England. A core libraries dataset is at the minimum of what the public should know about public libraries. Where are they? Where were the ones that closed? How many are there? What about the different types of library: Volunteer, Local authority, Commissioned.

This data doesn't answer any great questions about outcomes from library usage, the value of libraries, or provide any significant measures. Location data and counts of libraries are important though. A single location holds a great deal of associated information. Deprivation indices measure the relative deprivation of an area, under categories such as income, education, housing, and crime. Public Libraries in England are also tied to our local government administrations which themselves will have different geographical profiles. Some may be very rural widespread areas that require a different approach to a library service to one in a London Borough, or in a City authority.

This project is experimental, using various open datasets such as authority boundaries, population statistics, and the deprivation indices to merge these with the core libraries dataset and see what can be done.

It may not always be correct, and the methodology is likely to have plenty of issues, but it is open source. That means the full code is hosted on GitHub. A step-by-step description of how the underlying data was merged with other sources is given as part of that GitHub repository.

If you find any errors then shout about these, but also suggest improvements and features. For those with GitHub accounts you can raise bugs and feature requests (and get involved in the coding).

The core dataset is a start, at a national level, for English public libraries open data. As well as maintaining and improving that dataset it should be used as a first step to far more data. Loans, expenditure, footfall, PC usage, catalogue data, and more - for all library services in England. These are datasets that combined together could be used by the public to create real insight into library usage. There should be no need to prove the value of libraries, but there's plenty to gain from a better understanding of the data, and using that to improve services. For a great introduction to libraries open data please do look at (and copy) Newcastle Libraries open datasets and hack events.
