about

public library open data

Public library data is rich, varied, interesting, and there is a great demand for it. But with more public open data available than ever, data about libraries is still very rare.

Other than protecting citizen personal information, all data related to public libraries should be made openly licensed, and freely available for anyone to use. The Libraries Taskforce for England have released their core libraries dataset, compiled by the Taskforce and validated by the 151 library authorities in England. A core libraries dataset is at a minimum what the public should know about libraries. Where are they? Where were the ones that closed? How many are there? What about the different types of library, and how has that changed: volunteer, local authority, commissioned?

This data doesn't answer any great questions about outcomes of library usage on society, the value of libraries, or provide significant measures. But good location data and counts are important. A single location holds a great deal of associated information. Deprivation indices measure the deprivation of an area, under categories such as income, education, housing, and crime. Public Libraries in England are also tied to our local government administrations, all with different geographical profiles. Some may be rural widespread areas that require a different approach to a library service in a London Borough, or in a Unitary City authority.

this project

This project is experimental, using open datasets such as authority boundaries, population stats, and deprivation indices. These are merged with the core libraries dataset to provide alternative ways of exploring the data.

It may not always be correct, and the methodology is likely to have plenty of issues, but it is open source. This means the full code is hosted on GitHub. A step-by-step description of how the underlying data was merged with other sources is given as part of that code.

If you find any errors then shout about these, but also please suggest improvements and features. For those with GitHub accounts you can raise bugs and feature requests (and get involved in the coding).

The core dataset is a start, at a national level, for English public libraries open data. As well as maintaining and improving that dataset, it should be a first step to more data. Loans, expenditure, footfall, PC usage, catalogue data, and more - for all library services in England. These are datasets that could be used by the public to create real insight into library usage. There should be no need to prove the value of libraries, but there's plenty to gain from a better understanding of the data, and using that to improve services.

For a great introduction to libraries open data please do look at Newcastle Libraries open data and hack events.

comments on the dataset

The Libraries Taskforce dataset is not intended to be a finished product, it is a first version of a dataset that has a long way to go. But it requires investment in time by the library services, and those who want to use the data. Here are an initial set of commments on the current version.

Current field: Library serviceThe library authoritity e.g. Gloucestershire. Some issues with this field are the variety of ways of expressing the name, which may not match to official stats. For example, 'Bath and NE Somerset', where Bath and North East Somerset should probably be used.
Current field: Library nameStandardisation of naming conventions would be useful here. For example, some are appended with 'library', others not. Some have very descriptive names such as 'Anlaby Library and Customer Service Centre at East Riding Leisure Haltemprice' - some of that information may be more appropriate in the address details.
Suggested field: Unique Property Reference NumberThis suggested field would be the local authority Unique Property Reference Number. This identifier is a single reference that would allow cross matching of the address to various other local authority datasets. This reference would also allow for exact geo-coordinates of the location, taking these from Ordnance Survey data. This is not open data but there is a 'presumption to publish' for various government datasets - generally provided the information does not compromise OS commercial capabilities. A 3000 dataset of libraries would not seriously damage OS commercial property, so a request could likely be made to allow this data to be released with an open licence.
